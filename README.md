# Python Foundations Assignment
## Description
This repository contains solutions for three Python tasks:

### Level 1: Reverse a String
- Takes user input and reverses it using slicing.

### Level 2: Count Vowels
- Counts the number of vowels in a given string.

### Level 3: CLI Calculator
- A simple command-line calculator that performs:
  - Addition
  - Subtraction
  - Multiplication
  - Division
